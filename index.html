    NMAP

    Find how many hosts are active on a network?

    nmap -sP <network address>/<subnet>

    How to find which of the hosts above are acting as a web-server?
    • To act a web-server typically the server will have port 80,443 in open state

    Alert! nmap -sT -p 80,443 <network address>/<subnet>

    • T-> TCP Connect (Full open scan) 3-way handshake

    Now, using the above command might get you in trouble, IDS/ Firewall will detect that there is something wrong. To avoid that:-

    nmap -sS -p 80,443 <network-address>/<subnet>

    • S-> Stealthy (SYN Scan; Half-open)

    Rather than 3-way handshake it's just SYN,SYN-ACK

    How to find which OS is being used by the target server?

    nmap -O <server-ip>

    How to find the OS, version, scan script and traceroute?

    nmap -A <server-ip>

    Decoy (-D), using this will generate traffic not only from your laptop but from another src.ip which you mention in the command below, the destination won't know who is actually sending this traffic

    nmap -sS -D <Src.ip> <Dest.ip>
